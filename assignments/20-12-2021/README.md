## 3. Find Number is odd or even
```
1. START
2. PRINT "Enter a number: "
3. INPUT n
4. IF n % 2 EQUALS 0 THEN
        PRINT "even"
   ELSE
        PRINT "odd" 
5. STOP
```

## Print grade obtained by student applying following rules
### Rules
| Marks | Grade |
| --- | --- |
|  > 75 | O |
| 60-75 | A |
| 50-60 | B |
| 40-50 | C |
|  < 40 | D |

```
1. START
2. PRINT "Enter your marks: "
3. INPUT marks
4. IF marks > 75
        PRINT "O"
    ELSE IF marks > 60
        PRINT "A"
    ELSE IF marks > 50
        PRINT "B"
    ELSE IF marks > 40
        PRINT "C"
    ELSE
        PRINT "D"
5. STOP
```
